#  How to use
### 1. Fork
      - Fork the repo
### 2. Change file name
      - Rename the .html file of your choice to index.html
### 3. Turn on github pages
      - Go to the repo settings
      - Click Pages
      - Deploy from branch
      - Choose main
      - Save
### 4. Visit your site after waiting a bit
